
# ⚛️ CŒUR DE FORGE TTU — Streamlit Version

## Description
Application scientifique locale basée sur le modèle TTU-MC³.
Aucune API externe. Aucun appel IA.

## Fonctionnalités
- Upload TXT, PDF, DOCX, CSV, JSON
- Extraction locale
- Calcul ΦC, ΦD
- Seuil critique 0.5088
- Génération rapport téléchargeable

## Installation

```bash
git clone <your-repo>
cd forge_ttu_streamlit
pip install -r requirements.txt
streamlit run app.py
```
